To Do List

This is a mobile app written in Dart, using Flutter so that it can be installed on Android, iOS, and eventually Fuchsia.

At v1, it will allow for To-Do items with specific due dates and generic items. It will hopefully add the ability for optional additional notes and sublists within each item.

Language: Dart
