Below are a list of the languages and technologies I am knowledgeable in, a brief description of what they are and how they are useful in the field of Software and Web Development.

HTML5

HTML is the standard and most popular markup language used in developing web pages and web-based applications.

CSS3

CSS is a stylesheet language used for styling the visual aspect of a document written in a markup language, such as HTML or PHP.

JavaScript

JavaScript is a high-level, client-side programming language, most often used to create web applications or add dynamic elements in HTML-based web pages.

Java

Java is a general-purpose, object-oriented, statically typed programming language and is among the most popular and wide used computer-languages in the world.

Kotlin

Kotlin is very similar language to Java, it was created as a more modern alternative, though it still heavily relies on Java in that it runs on the JVM and uses Java Class Libraries.

Go

Go is a concurrent CSP style, general-purpose language developed by Google in an effort to replace C with a more modern and efficient replacement for the C language.

Python

Python is a dynamically typed, general-purpose, object-oriented, high-level language. Thanks to its vast number of libraries, it has been quickly rising in popularity in recent years.

SQL

SQL is a domain-specific language for data management and the most commonly used language in relation databases.

AWS

Amazon Web Services is cloud computing platform for individuals or companies to start a wide variety of servers on the fly, including the server that powers this site.

Windows

Windows is a graphical NT-based operating system mady by Microsoft, and is the most commonly used in personal computing.

Linux

Linux is an open-source operating system, with a wide variety of distributions. Because of this, it is heavily used in software development, and is the most popular OS used on servers.

macOS

macOS (previously Mac OS) is a graphical Unix-based operating system, used exclusively in Apple products, and is second to Windows in personal computing popularity.
