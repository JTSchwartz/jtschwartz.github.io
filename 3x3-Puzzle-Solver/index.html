3x3 Grid Puzzle Solver

This program was part of a CPS 350 Data Structures assignment. The program takes a picture, breaks it into a 3x3 grid, removes one piece, and shuffles them; from here the user can attempt to solve the puzzle themselves or have the program solve it at any point.

Base code and GUI were provided by the professor. The algorithm to solve the puzzle was written by myself using a Breadth-First Search algorithm.

Language: Java
