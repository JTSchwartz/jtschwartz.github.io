DNS Resolver

This program will determine the computers local DNS server, build and sent DNS Requests to the server, and interpret the first answer. It can create and send A and PTR records and can interpret both, as well as CNAME records. If an A record is requested and the answer contains a CNAME record, it will interpret the CNAME as well the first A record. The program can run in interactive mode or batch mode, for interactive, build and run the exe, and run it will a URL or IP as an argument. For batch mode, run the exe (with a dns-in.txt file in the directory) with a positive integer for the number of threads it should run with as an argument. dns-in.txt must have one URL/IP per line. Batch mode will then create a dns-out.txt.

Language: Python
