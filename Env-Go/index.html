Env

This program is an imitation of the Linux env tool, written in Go for a CPS 444 Linux Programming class. Without a command as an argument, the program will print out all current environment variables. Arguments are accept in the form of env variables and then a command and it's arguments, this will then run the command in a pseudo-environment with the new variables. Using the -i flag will force it to run with only the provided variables.

Language: Go
